牛肉とにんじんの煮物
材料: 牛肉, 根菜類（にんじん）
  1. にんじんは乱切り、牛肉は食べやすく切る。
  2. 鍋に油を熱し牛肉をさっと炒める。
  3. にんじんを加えて水・だしを入れ、煮る。
  4. 醤油・みりん・砂糖で味を調え、煮汁が少なくなるまで煮詰める。
牛肉とほうれん草の炒め物
材料: 牛肉, 葉物野菜（ほうれん草）
  1. ほうれん草は茹でて水気を切り、食べやすく切る。
  2. 牛肉を炒め、塩胡椒で下味をつける。
  3. ほうれん草を加え、醤油少々で調える。
  4. 火を止めてごま油を垂らして香り付け。
牛肉とトマトの炒め物
材料: 牛肉, 果菜類（トマト）
  1. トマトは角切り、牛肉は薄切りにする。
  2. 牛肉を炒め、トマトを加えて炒め合わせる。
  3. 塩、黒胡椒、好みでケチャップ少々で味を整える。
  4. 仕上げにバジルやパセリを散らすと爽やか。
牛肉としいたけのバター炒め

  3. 牛肉を加えて炒め、塩胡椒で味付け。
  4. 仕上げに醤油を少量回して香ばしく。

  1. まいたけを炒め、枝豆を加える。
  2. 卵を流し入れてふんわりと仕上げる。

  2. オリーブオイルと塩胡椒で和えて完成。
